# music

Modular Faust instruments and routing components for building evolving JACK and PipeWire performance systems. Each patch is designed as a composable node, so the repository can grow from single voices into larger playable networks without rewriting its fundamental architectural assumptions.

## Kick Suite Diagnostics

For kick-suite audio degradation investigations, start with `kick-suite/DIAGNOSTICS.md`, then read `kick-suite/.runlogs/current-summary.md`.
